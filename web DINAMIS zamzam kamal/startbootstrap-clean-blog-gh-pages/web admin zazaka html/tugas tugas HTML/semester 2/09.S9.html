Biasanya pada penggunaan rumah pada umumnya, layanan internet terintegrasi dengan telepon rumah. Untuk merubah sinyal telepon, maka digunakanlah modem ADSL sebagai “Converternya” untuk diteruskan ke perangkat jaringan lainnya. </p></h4>

    
    <h2> SWITCH </h2>
              <center><img src="images/switch.jpg"></center>
            <i><h5><center> Gambar 1.6 Switch </center><h5></i>
    <h3><p> switch Switch merupakan “terminal” untuk mengubungkan satu sama lain perangkat yang terhubung dalam sebuah jaringan untuk kemudian diteruskan agar dapat saling berkomunikasi data. Switch bekerja pada layer 2 (data link) dari model OSI (Open System Interconection). Switch terdapat 2 jenis, yaitu switch manageable atau switch yang dapat dikonfigurasi fiturnya, dan Swtich Unmanaged yaitu yang secara fungsinya hanya mengandalkan fitur bawaan pabrik. </p></h3>


    <h2> ROUTER </h2>
    <h3><p> Router adalah sebuah perangkat yang berfungsi untuk meneruskan paket – paket dari sebuah network ke network yang lainnya (baik LAN ke LAN atau LAN ke WAN) sehingga host-host yang ada pada sebuah network bisa berkomunikasi dengan host-host yang ada pada network yang lain. Router menghubungkan network – network tersebut pada network layer dari model OSI, sehingga secara teknis Router adalah Layer 3 dan berfungsi biasanya sebagai Gateway. </p></h3>

    <h2> NETWORK INTERFACE CARD (NIC) </h2>
              <center><img src="images/nic.png"></center>
            <h5><center> Gambar 1.8 NIC </center><h5>
    <h4><p> Network Interface Card (NIC) merupakan perangkat wajib yang harus dimiliki oleh semua host (Komputer) agar dapat terhubung kedalam sebuah jaringan. NIC menyediakan akses ke media fisik jaringan. Tegangan listrik, arus, gelombang elektromagnetik dan besaran satuan fisik lainnya dibentuk dan ditentukan oleh NIC. NIC bekerja pada layer 1 model OSI atau disebut juga layer physical. </p></h4>


    <h2> ACCESS POINT </h2>
              <center><img src="images/AP.jpg"></center>
            <i><h5><center> Gambar 1.9 Access Point </center><h5></i>
    <h4><p>Access Point adalah perangkat jaringan yang berfungsi sebagai media penghantar sinyal wireless agar dapat ditangkap oleh client yang memiliki fungsi wireless. Fungsinya sama seperti layaknya perangkat switch hanya saja Access Point berjalan pada mode wireless atau nirkabel. </p></h4>

Dengan kata lain perangkat ini dapat disebut juga “Router yang memiliki fitur wireless”. </p></h3>
