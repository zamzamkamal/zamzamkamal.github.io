Pengertian Link


HTML link atau hyperlink, adalah sekumpulan kata atau gambar yang apabila di klik akan menuju ke dokumen lain atau halaman lain atau bagian lain atau halaman yang sama

Macam-macam kink di dalam HTML :
  • Link Relatif
  • Link Absolut
  • Link ke bagian lain dalam dokumen yang sama
  • Link ke E-mail

1. Link Relatif

Link Relatif dibuat apabila kita membuat sesuatu page ke page yang lain pada komputer yang sama, tidak memerlikan alamat internet yang lengkap. Contoh kita akan membuka file hasil latihan Unordered List.

2. Link Absolut

Link Absolut dibuat apabila kita akan membuat link ke page web lain yang berada di luar web site kita sehingga alamat yang kita tulis harus lengkap sesuai dengan alamat web yang akan kita tulis. Contoh kita akan membuat link ke web Google

3. Link E-Mail

Link email digunakan untuk membuat link saat kita akan membuat suatu email(tidak bisa jika aplikasi email tidak terinstal). contoh kita akan mengirim email ke felisyaus@gmail.com
