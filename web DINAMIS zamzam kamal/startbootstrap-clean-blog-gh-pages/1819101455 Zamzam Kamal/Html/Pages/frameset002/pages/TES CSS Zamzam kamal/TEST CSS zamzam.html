Perangkat Jaringan Dasar

MODEL ADSL

Gambar 1.5 Modem ADSL

Modem (Modulator Demudelator) Merupakan Perangkat yang secara fungsinya adalah megubah sinyal digital menjadi sinyal analog dan sebaliknya. Modem ADSL (Asymmetric Digital Subscriber Line) adalah perangkat yang digunakan untuk menghubungkan komputer atau router ke saluran telepon. Biasanya pada penggunaan rumah pada umumnya, layanan internet terintegrasi dengan telepon rumah. Untuk merubah sinyal telepon, maka digunakanlah modem ADSL sebagai “Converternya” untuk diteruskan ke perangkat jaringan lainnya.

SWITCH

Gambar 1.6 Switch

switch Switch merupakan “terminal” untuk mengubungkan satu sama lain perangkat yang terhubung dalam sebuah jaringan untuk kemudian diteruskan agar dapat saling berkomunikasi data. Switch bekerja pada layer 2 (data link) dari model OSI (Open System Interconection). Switch terdapat 2 jenis, yaitu switch manageable atau switch yang dapat dikonfigurasi fiturnya, dan Swtich Unmanaged yaitu yang secara fungsinya hanya mengandalkan fitur bawaan pabrik.

ROUTER

Router adalah sebuah perangkat yang berfungsi untuk meneruskan paket – paket dari sebuah network ke network yang lainnya (baik LAN ke LAN atau LAN ke WAN) sehingga host-host yang ada pada sebuah network bisa berkomunikasi dengan host-host yang ada pada network yang lain. Router menghubungkan network – network tersebut pada network layer dari model OSI, sehingga secara teknis Router adalah Layer 3 dan berfungsi biasanya sebagai Gateway.

NETWORK INTERFACE CARD (NIC)

Gambar 1.8 NIC

Network Interface Card (NIC) merupakan perangkat wajib yang harus dimiliki oleh semua host (Komputer) agar dapat terhubung kedalam sebuah jaringan. NIC menyediakan akses ke media fisik jaringan. Tegangan listrik, arus, gelombang elektromagnetik dan besaran satuan fisik lainnya dibentuk dan ditentukan oleh NIC. NIC bekerja pada layer 1 model OSI atau disebut juga layer physical.

ACCESS POINT

Gambar 1.9 Access Point

Access Point adalah perangkat jaringan yang berfungsi sebagai media penghantar sinyal wireless agar dapat ditangkap oleh client yang memiliki fungsi wireless. Fungsinya sama seperti layaknya perangkat switch hanya saja Access Point berjalan pada mode wireless atau nirkabel.

WIRELESS ROUTER

Gambar 1.10 Wireless Router

Wireless Router adalah perangkat yang mirip dengan Access Point, hanya saja pada wireless router, perangkat ini memiliki fungsi lain seperti Routing, Network Address Translation (NAT), dan fungsi – fungsi yang dimiliki perangkat router. Dengan kata lain perangkat ini dapat disebut juga “Router yang memiliki fitur wireless”.
